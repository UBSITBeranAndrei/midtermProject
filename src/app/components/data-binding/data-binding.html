Data Binding Examples

All Angular data binding concepts demonstrated in one place.

01

Interpolation {{ '{{' }} value {{ '}}' }}

Displays component data directly in the template using double curly braces.

Student: {{ studentName }}

Course: {{ course }}

Year: {{ year }}

Expression: {{ year + 1 }}

Uppercase: {{ studentName.toUpperCase() }}

{{ '{{' }} studentName {{ '}}' }} {{ '{{' }} year + 1 {{ '}}' }} {{ '{{' }} studentName.toUpperCase() {{ '}}' }}
02

Property Binding [property]

Binds component properties to HTML element attributes using square brackets.

[disabled]="isDisabled" [src]="imageUrl" [placeholder]="buttonLabel"
03

Event Binding (event)

Listens to DOM events and triggers component methods using parentheses.

Last Key Pressed: {{ lastKey }}

{{ isHovered ? '🔥 Hovering!' : '👆 Hover over me' }}
(click)="onButtonClick()" (keyup)="onKeyPress($event)" (mouseenter)="onMouseEnter()"
04

Two-way Binding [(ngModel)]

Syncs data between the component and template in both directions simultaneously.

Name: {{ twoWayName || '—' }}

Email: {{ twoWayEmail || '—' }}

Age: {{ twoWayAge || '—' }}

[(ngModel)]="twoWayName" [(ngModel)]="twoWayEmail" [(ngModel)]="twoWayAge"
05

Angular Signals signal()

Angular 21's reactive primitive for state management. Signals automatically track dependencies.

{{ counter() }}

{{ signalMessage() }}

counter = signal<number>(0) counter.update(v => v + 1) counter.set(0) {{ '{{' }} counter() {{ '}}' }}
06

Class & Style Binding [class] [style]

Dynamically applies CSS classes and inline styles based on component state.

{{ isActive ? '✅ Active Class Applied' : '❌ No Active Class' }}

This text is dynamically styled!

[class.box-active]="isActive" [style.fontSize.px]="fontSize" [style.color]="selectedColor"
