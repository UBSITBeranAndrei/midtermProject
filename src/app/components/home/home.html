Supplier Management

View all suppliers in a table. Click View to see full supplier details, edit information, and save changes with two-way data binding.

⚡

Angular Signals

This project uses Angular 21's latest signal-based state management for reactive, efficient data handling across components.

🔗

Routing & Navigation

Parameterized routes allow direct navigation to supplier details. The router persists service data across page transitions.

🔒

Session Auth

Basic session authentication using sessionStorage. Login and logout functionality is demonstrated in the Products section.

💾

Shared Service

A centralized ProductService and SupplierService manage all data. Updates in one component instantly reflect across the entire app.
